# 汇编语言第一次大作业——快速排序

## 实验内容

- 请根据文件`partition.s`以及`quicksort.s`中的`partition_c`y和`quicksort_c`函数, 完善相应的`partition_asm`和`quicksort_asm`函数, 并通过测试. 可以使用`x86-64`, `ARM64`, `LoongArch`, `RISC-V` 等汇编语言.
